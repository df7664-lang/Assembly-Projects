# Assembly Projects - Two-Pass Assembler

This repository contains a professional implementation of a **Two-Pass Assembler** written in C. 

## Features
* **First Pass:** Symbol table management and instruction analysis.
* **Second Pass:** Generating machine code and output files (`.ob`, `.ent`, `.ext`).
* **Macro Expansion:** Pre-assembler stage for processing macros.
* **Memory Management:** Efficient handling of dynamic data structures.

## How to Run
1. Clone the repository.
2. Compile using the provided Makefile:
   ```bash
   make

./assembler file1 file2
